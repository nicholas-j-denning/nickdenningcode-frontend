nickdenningart.com

I made nickdenningart.com as a gallery for my algorithmic fractal artwork. Users can browse by tags and download high resolution versions or follow a link to buy a print in my Shopify store. The application also has an admin page which I use to upload new art. When I do, it also creates a product with my dropshipping print provider Printify using their HTTP API to make prints of that piece available in my store.

The frontend is written with Angular and Bootstrap for a responsive experience that works well on mobile or different screen sizes. It's hosted in S3 behind CloudFront for fast worldwide availability.

The backend is written in Java with Spring Boot and hosted on AWS Lambda.

Because I used a completely serverless scale-to-zero architecture with Lambda, S3, and DynamoDb, the app costs less than $2/month to run, with the largest cost being the domain name. It also can automatically scale to any number of users, even millions.

All infrastructure is provisioned as code using AWS Java CDK, including full CI/CD pipelines for the frontend and backend using CodeBuild.
